## Переименование скриншотов

Переименовывает файлы `Screenshot_*.png` в формат screen_YYYY_MM_DD.png используя дату создания файла

## Польза

[prntscr](https://app.prntscr.com/ru/) сохраняет скриншоты на рабочий стол с неймингом `Screenshot_*.png`. Я их скидываю в папку sort, и когда приходит время, там много файлов с одинаковыми названиями. Самое время их переименовать

## Когда не пригодится

У всех пользователей MacOS скриншоты называются «Снимок экрана хх.хх.хххх» — им не нужды.

## Требования

Необходим php

## Использование

```sh
php -f script.php
```
