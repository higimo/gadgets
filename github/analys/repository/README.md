## Выгрузить репозитории со страницы

Работает в консоли браузера

## Польза

Понять, какие есть репозитории, какие из них приватные, какое у них описание и оценить консистентность названий

Например, я унифицировал названия до `[api|tg|back|front]-[site]-[ts|js|php]`
Например, я унифицировал описания, для всего неготового добавил «`[alpha]`», указал основную технологию: md|react|laravel|php|C#

## Как запускать

1. Открыть список репозиториев https://github.com/higimo?tab=repositories
2. Открыть консоль браузера `CTRL` + `F12`
3. Вставить скрипт
4. Скопировать JSON

## Что делать дальше

Скрипт даёт JSON. Можно работать в текстовом редакторе. А можно обернуть в таблицу и работать в редакторе. Вот скрипт для трансляции в табилцу

```js
[
    Object.keys(temp1[0]),
    ...temp1.map(Object.values)
].map(r=>r.join('\t')).join('\n')
```

Вывод объекта можно сохранить прямо в консоле в глобальную переменную, по умолчанию она называется `temp1`. По необходимости, её надо переименовать в скрипте
