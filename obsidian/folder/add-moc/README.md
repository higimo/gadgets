## Добавить MOC в каждый файл в папке

Работает рекурсивно, сам ищет MOC файл.

MOC-файл обязан называться как папка, в которой он находится

## Польза

Когда граф выглядит как несвязанные ноды, этот скрипт автоматически проставит MOC везде, где потребуется

## Как запускать

Первым аргументом нужно указать папку в хранилище

```sh
php -f obsidian/folder/add-moc/script.php /obsidian-vault/Блог
```
